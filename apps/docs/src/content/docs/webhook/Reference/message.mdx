This event is sent when a client sends a message through the Websocket.

## Data Schema

```ts
{
  // ...
  "data": {
    "connection": {
      "channel": string;
      "projectClientId": string;
      "projectId": number;
      "identifier": string;
      "data": string; // This is the data you sent when creating the connection
    }
    "message": string;
  }
}
```

## Response

You can respond to this event to send a message to the client.

- Code `200`: If the response contains a body, it will be sent to the websocket.
- Code `204`: Nothing will be sent to the client.
