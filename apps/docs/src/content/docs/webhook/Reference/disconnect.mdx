This event is sent when a client disconnections from a Websocket (supposing it connected successfully before).

## Data Schema

```ts
{
  // ...
  "data": {
    "connection": {
      "channel": string;
      "projectClientId": string;
      "projectId": number;
      "identifier": string;
      "data": string; // This is the data you sent when creating the connection
    }
    "code": number;
    "reason": string;
    "clean": boolean;
  }
}
```
